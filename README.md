# Grover Search

Material de hands-on sobre o algoritmo de Grover e portas logicas quanticas.

## Conteudo

- `grover_hands-on..ipynb`: notebook principal com exercicios de portas quanticas, simulacao e busca com Grover.

## Objetivos do hands-on

1. Verificar a acao de portas logicas quanticas no estado dos qubits.
2. Plotar a quantidade de iteracoes em funcao de `N` e observar a complexidade quadratica.
3. Usar o algoritmo de Grover para encontrar elementos de uma lista de compras que comecam com a letra `A`.

## Ambiente

Crie um ambiente Python e instale as dependencias:

```bash
python3 -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
```

Depois abra o notebook:

```bash
jupyter notebook
```
